ONNX Model Download Instructions
================================

This demo requires the ResNet50 v2 ONNX model for image classification.

Download URL:
https://github.com/onnx/models/raw/main/validated/vision/classification/resnet/model/resnet50-v2-7.onnx

File size: ~98 MB

After downloading, place the file in this directory:
Assets/model/resnet50-v2-7.onnx

Optional: Download ImageNet labels file for human-readable class names:
https://raw.githubusercontent.com/pytorch/hub/master/imagenet_classes.txt

Save as: Assets/model/imagenet_labels.txt
